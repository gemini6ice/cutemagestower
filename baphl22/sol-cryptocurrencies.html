Cryptocurrencies Solution

by Ata Gurpinar

Each entry in the boxes is a common money-related phrase. Unlike a traditional letter links puzzle, the connected letters from one entry to the next are not the same letter, but instead are all mappings from a shared alphabetic cipher (consistent with the directionality of the arrows). Thus, any boxes in the same entry that have the same arrow type and direction attached to it are also the same letter. All pairs of letters that are present between two entries that are a mapping in the shared cipher are given arrows. The graduate puzzle provided the initials of each word, and the undergraduate version provided every letter that was not linked.

The entries are formed by the trigrams provided on the second page, with each stylized letter being the enciphering of the letter actually in the entry. Between the enciphered letters in the trigrams and the cipher mappings from the diagram, the given template for the cipher can be filled out as follows:

ABCDEFGHIJKLMNOPQRSTUVWXYZ
SUMFINALTWOZYXVR JKPQ DC

Although the cipher is not completely determinable from the entries and trigrams, the beginning is, and it spells the clue phrase SUM FINAL TWO. Since the whole number in each trigram value was used for ordering, "final two" refers to the two digits after the decimal point. Each entry's values add up to a number from 1 to 26, which stand in for letters. However, the letters are enciphered and the cipher from before must be used to decipher them to a final answer:
